Lógica de programação com javascript - WHILE - Exercício

Escreva um programa que solicita ao usuário que adivinhe um número aleatório entre 1 e 100. O programa deve continuar pedindo que o usuário adivinhe até que ele acerte o número. O programa deve exibir uma mensagem informando se o número digitado pelo usuário é maior ou menor do que o número aleatório. Exemplo: O programa gera o número aleatório 57. Se o usuário digitar 50, o programa deve exibir "O número é maior". Se o usuário digitar 60, o programa deve exibir "O número é menor". O programa deve continuar pedindo que o usuário adivinhe até que ele acerte o número 57.
