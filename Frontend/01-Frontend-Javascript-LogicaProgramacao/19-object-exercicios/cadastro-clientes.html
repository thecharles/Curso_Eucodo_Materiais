Lógica de programação com javascript - Cadastro de imóveis

Escreva um programa em javascript que funcione como um cadastro de imóveis e atenda aos seguintes requisitos: - Deve ter um menu interativo que sempre é exibido até que o usuário escolha sair. - O menu deve ter a opção de salvar um imóvel - Para salvar um novo imóvel, o programa deve pedir as seguintes informações: Nome do proprietário, qtde de quartos, qtde de banheiros e se possui garagem (sim ou não). - O menu também deve ter a opção de mostrar todos os imóveis salvos.
