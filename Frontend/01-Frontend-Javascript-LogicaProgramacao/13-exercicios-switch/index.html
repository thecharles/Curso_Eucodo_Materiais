Exercícios - SWITCH

Escreva um programa em JavaScript que solicita ao usuário o nome de um mês e exibe o número de dias desse mês. Use o comando switch para verificar qual é o mês e exibir o número correto de dias. Exemplo: Se o usuário digitar "fevereiro", o programa deve exibir "28 dias". Se o usuário digitar "abril", o programa deve exibir "30 dias".
