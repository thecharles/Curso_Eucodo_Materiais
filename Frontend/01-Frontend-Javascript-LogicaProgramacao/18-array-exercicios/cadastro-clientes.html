Lógica de programação com javascript - Cadastro de clientes

Escreva um programa em javascript que solicita o nome do cliente e acrescente (push) ele numa lista lista. Após acrescentar, exibir essa lista na tela com os clientes e suas respectivas posições no array (índice). Dê a opção para o usuário remover o cliente se ele quiser pedindo o índice e removendo do array. Peça a opção via prompt().
